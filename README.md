## 🚀 How to Deploy on GitHub Pages

1. **Create a new GitHub repository**
   - Go to [github.com](https://github.com) → Click **New Repository**
   - Name it: `corpolite-website` (or any name you prefer)
   - Set visibility to **Public**
   - Click **Create Repository**

2. **Upload the files**
   - Click **uploading an existing file** on the repo page
   - Drag and drop `index.html`, the `css/` folder, and `README.md`
   - Click **Commit changes**

3. **Enable GitHub Pages**
   - Go to your repo → **Settings** → **Pages** (left sidebar)
   - Under **Source**, select `Deploy from a branch`
   - Choose branch: `main` | folder: `/ (root)`
   - Click **Save**

4. **Your site is live!**
   - After ~1–2 minutes, your site will be live at:
   - `https://YOUR-USERNAME.github.io/corpolite-website/`
